- springBoot + mybatis + thymeleaf <br>

## 🖥️ 프로젝트 소개
헬스장 및 운동 시설의 고객관리 프로젝트입니다. <br>
오래된 헬스장에 가보면 수기로 작성하여 회원을 관리하고 있어서 <br>
기본적인 회원 관리 및 매출현황을 보여드리고자 프로젝트를 만들었습니다.
